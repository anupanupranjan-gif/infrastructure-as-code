# DevOps & SRE Learning Labs

This repo contains hands-on labs for practicing DevOps and SRE skills.

## Labs included

- **devops-terraform**: Terraform experiments for infrastructure provisioning
- **devops-ansible**: Ansible playbooks and automation exercises
- **devops-kubernetes**: Kubernetes deployments and management
- **sre-observability**: Prometheus, Grafana, and monitoring exercises

Each lab folder contains a README explaining its purpose, setup, and exercises.
